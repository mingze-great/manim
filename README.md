# 思维可视化内容平台

这是一个统一的内容创作平台，当前包含三个模块：

- 思维可视化：通过对话生成讲解脚本，再生成可视化视频
- 火柴人视频：支持一键或分步生成分镜、图片、配音与视频
- 公众号文章：支持轻量版快速出稿和专业版精修图文排版

## 平台入口

- 左侧 `开始创作`：所有创作模块的统一入口
- 左侧 `我的作品`：统一查看思维可视化、火柴人视频和公众号文章的历史作品
- 左侧 `帮助中心`：查看操作教程和使用说明

注意：公众号文章不再单独放在左侧导航中，避免与 `开始创作` 重复。

## 权限说明

- 普通用户默认只开放 `思维可视化`
- `火柴人视频` 与 `公众号文章` 需要管理员在后台单独开通
- 管理员账号三个模块默认无限制

## 三个模块怎么用

### 1. 思维可视化

适合：知识讲解、思维模型、数学推导、概念拆解

推荐流程：
1. 进入 `开始创作`
2. 选择 `思维可视化`
3. 选择热门方向或直接输入主题
4. 在对话页补充要求，逐步打磨内容
5. 生成脚本与视频，进入任务页查看渲染进度

### 2. 火柴人视频

适合：口播讲解、观点表达、人物叙事、情绪故事

推荐流程：
1. 进入 `开始创作`
2. 选择 `火柴人视频`
3. 选择热门主题或手动输入主题
4. 选择音色、语速、配音来源
5. 可选上传风格参考图
6. 选择一键生成或分步创作
7. 生成图片、配音并合成视频

### 3. 公众号文章

适合：公众号运营、知识图文、自媒体图文输出

推荐流程：
1. 进入 `开始创作`
2. 选择 `公众号文章`
3. 选择热门方向或手动输入主题
4. 进入轻量版或专业版
5. 生成文案、配图、排版
6. 在手机预览中确认图文顺序
7. 复制图文内容到公众号编辑器

## 轻量版与专业版

### 公众号轻量版

适合快速出稿：
- 先生成文案草稿
- 可直接修改标题和正文
- 生成配图并调整插入位置
- 生成排版并复制图文内容

### 公众号专业版

适合精修：
- 大纲确认
- 正文精修
- 配图管理
- 手机预览
- 富文本复制

## 后台管理

管理员后台当前支持：

- 单用户模块权限设置
- 批量模块权限设置
- 模块权限模板
- 模块统计看板
- 用户详情（最近作品、最近任务、模块使用情况）
- 公众号分类配置
